### Setup to run in GitHub


1.
```bash
dx bundle --release --out-dir docs
```

2. Take all item in public to "docs"

3. Drop "docs" folder directly to GitHub drop file zone

4. In your repo, -> Settings -> Pages

5. Follow below image

![image](https://github.com/Chssam/underfine_runner/blob/main/Showcase/Screenshot%202026-05-26%20111320.png?raw=true)

6. Click save

7. Wait for it to deploy

8. Profit
